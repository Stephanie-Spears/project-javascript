Aritmetic

  • Web browsers can only do three things: display HTML content, style it with CSS, and change what'd displayed with JavaScript
  • ctrl+shift+j on a webpage will direct you to the JavaScript developer tools
  • JavaScript can perform basic aritmetic(add multipy, modulo, PEMDAS, etc.)
  • You can use +=, -=, *=, /=

Variable Naming

  • Variables should begin with a letter.
  • Variables are case sensitive
  • use clear names that describe the value being stored
  • "var myNumber = 45;" sets myNumber to 45.
  • myNum type naming is called lower camel case--because the caps look like humps
  • you should always use var myNum when declaring a variable to avoid bugs
  • increment and decrement by one: myNum += 1 or -= 1

STRINGS

  • if you want to put a quote inside a string you can do forward slash on either side of it, or use an "escape" which is two forward slashes. You can also use single quotes

    Functions

    • parseInt(): converts a string to a number
    • toString(): converts a number to a string
    • alert("") opens a dialog box and returns undefined
    • confirm() opens a dialog box and returns boolean--so can ask yes or no question
    • prompt("") opens a dialog box and returns a string
    • ALL values returned from a prompt are saved as strings, to collect a num, we need to run the .parseInt or .parseFloat function on it
    • an ARGUMENT is what you pass into a function or method, a PARAMETER is a variable that's assignmed to the argument. Below, number1 and number2 are PARAMETERS, and 1 and 2 are ARGUMENTS
